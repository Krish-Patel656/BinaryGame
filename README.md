# Binary Game

## Description
Binary Game is a simple console-based game where players convert decimal numbers to binary or vice versa. It helps improve your understanding of binary numbers and challenges your quick thinking skills.

## How to Play
1. The game will display a random decimal or binary number.
2. Enter the corresponding binary or decimal value.
3. The game checks your answer and gives feedback.
4. Earn points for each correct answer.

## Features
- Randomly generated decimal numbers.
- Real-time answer checking.
- Score tracking.

## Requirements
- [MARS MIPS Simulator](http://courses.missouristate.edu/kenvollmar/mars/)
- Basic understanding of MIPS I/O and arithmetic instructions.

## How to Run
1. Open `main.asm` in MARS.
2. Assemble the program.
3. Run the program using the MARS Run menu or toolbar.
4. Enter binary or decimal values as prompted.
